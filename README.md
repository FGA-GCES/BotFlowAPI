# Rasa Nlu Trainner API

## Run project
### Docker (recomended)

``docker-compose up --build``

### Via npm (or yarn)
*Node version: 10 or higher*

Inside folder server run:

``npm install``

or 

``yarn install``

Then run:

``npm start``

or

``yarn start``

The server will be avalaible in localhost on port 3030
